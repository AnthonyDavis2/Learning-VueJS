#### Directives 
- **v-bind (:)**: Dynamically binds one or more attributes, or a component prop to an expression.
- **v-model**: Creates two-way data binding on form input elements or components.
- **v-on (@)**: Attaches event listeners to the element.
- **v-if / v-else-if / v-else**: Conditionally renders elements based on the truthiness of the expression.
- **v-show**: Toggles the element's display CSS property based on a condition.
- **v-for**: Renders an element or template block multiple times based on a source data array.
- **v-html**: Updates the element's innerHTML.
- **v-text**: Updates the element's textContent.
- **v-pre**: Skips compilation for this element and all its children.
- **v-once**: Renders the element and component only once.
- **v-memo**: Memoizes a sub-template, skipping virtual DOM diffing if dependencies haven't changed.
- **v-cloak**: Hidden until the Vue instance has finished compiling.
- **v-slot (#)**: Used in components to define named slots or receive scoped slot props.
